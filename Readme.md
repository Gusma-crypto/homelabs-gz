# 🚀 My Learning Homelab & IT Journey

Selamat datang di repositori infrastruktur IT saya! Repositori ini adalah catatan perjalanan belajar saya dalam mengelola server, jaringan, dan layanan mandiri (*self-hosted*). Di sini saya menyimpan konfigurasi Docker, catatan instalasi Proxmox, dan berbagai eksperimen teknis lainnya.

> **Status:** 🛠️ *Work in Progress* — Saya sedang membangun lab yang stabil sambil mendalami dunia Virtualisasi dan Web3 Development.

---

## 📂 Daftar Isi

* **[Virtualisasi](proxmox/)**: Konfigurasi Proxmox VE, manajemen VM (CentOS/Ubuntu), dan LXC Container.
* **[Docker Stacks](apps/)**: Kumpulan file `docker-compose` untuk layanan mandiri (Self-hosted).
* **[Web & Web3](dev/)**: Lingkungan uji coba untuk proyek React, Rust, dan Smart Contracts.
* **[Networking](networking/)**: Pengaturan Mikrotik, instalasi Starlink, dan manajemen domain lokal.
* **[Monitoring](monitoring/)**: Pantauan performa server menggunakan dashboard sederhana.

---

## 💻 Hardware Setup

Saat ini saya fokus pada perangkat yang efisien untuk menunjang aktivitas belajar dan operasional bisnis IT saya.

### Compute & Storage

| Device | OS | Role | Specs (Target/Current) |
| :--- | :--- | :--- | :--- |
| **Main Server** | Proxmox VE | Virtualization Host | Intel Xeon / Core i5, 32GB+ RAM |
| **Dev Machine** | Linux / Win | Coding & Daily Driver | High Performance Workstation |
| **Storage** | Debian/Ubuntu | File Server & Backup | SSD for Boot, HDD for Data |

### Networking & Security

| Device | Description |
| :--- | :--- |
| **Starlink/ISP** | Sumber koneksi utama untuk kebutuhan bandwidth tinggi. |
| **Mikrotik** | Pengaturan bandwidth, VLAN, dan keamanan jaringan lokal. |
| **CCTV System** | Integrasi keamanan fisik yang dikelola secara mandiri. |
